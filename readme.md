## The Elm Camp website

https://elm.camp

Features:

- Static event info
- Dynamic inventory & ticket sales management
- Stripe integration

### Local development:

- [Install lamdera](https://dashboard.lamdera.app/docs/download)
- Run `lamdera live`

#### Troubleshooting
- To fix `resource busy (file is locked)` error, run `lamdera reset`
